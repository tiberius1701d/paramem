You are given a list of extracted facts. Each fact has a subject, predicate, and object. Your task is to find same-(subject, object) groups where multiple predicates express the SAME relationship — synonym predicates that should collapse into one.

RULES:
1. Only group facts that share the SAME subject AND the SAME object (exact match).
2. Within a group, cluster predicates that are genuine synonyms (express the same real-world relationship). Each cluster may have 1 or more predicates.
3. A cluster with only 1 predicate means that predicate has no synonym in this group — it is distinct and kept as-is.
4. NEVER merge predicates that are asymmetric (e.g. "manages"/"reports_to", "parent_of"/"child_of", "taught"/"studied_under") — those describe different directed relationships.
5. NEVER merge predicates when the objects differ — same subject, different object = two distinct facts, NOT synonyms.
6. If a group has only one predicate (no candidate synonyms), output it with a single-element cluster.
7. Do not invent subjects, predicates, or objects. Only use what appears in the input.
8. Output valid JSON only. No prose, no markdown fences.

In the examples below every JSON literal uses double braces {{ }} so the format engine collapses them to single braces. The output you emit must use plain single-brace JSON.

POSITIVE example — synonym predicates on the same (subject, object) pair cluster together; distinct facts are kept separate:
Facts:
[
  {{"subject": "Alex", "predicate": "works_for", "object": "Acme"}},
  {{"subject": "Alex", "predicate": "employed_by", "object": "Acme"}},
  {{"subject": "Alex", "predicate": "resides_in", "object": "Portland"}},
  {{"subject": "Alex", "predicate": "lives_in", "object": "Portland"}},
  {{"subject": "Alex", "predicate": "likes", "object": "hiking"}}
]
Output:
{{"groups": [
  {{"subject": "Alex", "object": "Acme", "clusters": [["works_for", "employed_by"]]}},
  {{"subject": "Alex", "object": "Portland", "clusters": [["resides_in", "lives_in"]]}},
  {{"subject": "Alex", "object": "hiking", "clusters": [["likes"]]}}
]}}

Facts:

  {{"subject": "Sam", "predicate": "lives_in", "object": "Berlin"}},
